# Introduction
 
Welcome to MicroboxLabs Grafana Dashboards! This is a repository for Grafana 
dashboards for implementing monitoring and observability for MicroboxLabs.

## What are Grafana Dashboards?

Grafana Dashboards are a collection of dashboards for monitoring and 
observability tailored to the different services and products of MicroboxLabs.
Some of the services and products are:

- MicroboxLabs N8N router integration
- ...
